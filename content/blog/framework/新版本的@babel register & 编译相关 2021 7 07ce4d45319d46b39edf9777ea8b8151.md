# 要安装的包

```bash
npm i @babel/register
npm i @babel/core
npm i @babel/preset-env
```

# 入口文件

```bash
require("@babel/register")({
	presets: ['@babel/preset-env']
});
require('./test.js')
```

# 入口是ts文件

```jsx
require("@babel/register")({
	presets: ["@babel/preset-typescript", '@babel/preset-env'],
});
```
