使用npm上发布自己的包

# npm 存全局的地方
/usr/local/lib/node_modules

注册一个npm账号

npm

使用终端登录

npm login

在现有项目下发布

npm publish
npm publish --access=public

首次发布之后要递增版本号

要求Git暂存区没东西

npm version patch
