    react 服务端渲染 & next.js使用

      reactssr

      📚 目录

        • next.js

      一般用来做官网，方便搜索引擎爬。

      优点

      • 网页的快速初始加载，因为准备好显示 HTML 被提供给浏览器。
      • 网页的内容被更快地编入索引，从而获得更好的 SEO 排名。

      缺点

      • SSR 需要更多资源并且可能很昂贵，因为所有处理都在服务器上完成。
      • 对于复杂的应用程序，大量的服务器请求会降低站点的速度。
